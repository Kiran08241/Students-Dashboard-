# 🎓 Students Dashboard
## 📌 Project Title
Student Performance Analytics Dashboard

## 📰 Headline
An Interactive Power BI Dashboard for Analyzing Student Academic Performance, Gender-wise Trends, Subject Scores, and Parental Education Impact.

## 📝 Short Description
Developed an interactive Power BI dashboard to analyze student scores in Math, Reading, and Writing. The dashboard includes KPI cards, gender-wise analysis, parental education insights, and interactive filters to support data-driven decision-making.

## 🛠 Tech Stack
- Power BI Desktop
- Power Query (Data Cleaning & Transformation)
- DAX (Measures & Calculations)
- Microsoft Excel / CSV Dataset
- Data Modeling

## 📂 Data Source
Kaggle – Students Performance in Exams Dataset (CSV)

## ✨ Features
- Interactive KPI Cards
- Subject-wise Performance Analysis
- Gender-wise Score Comparison
- Parental Education Level Analysis
- Ethnicity-wise Data Filtering
- Interactive Slicers
