# Students-Dashboard-
**Project Title:**
Student Performance Analytics Dashboard
**Headline:**
An Interactive Power BI Dashboard for Analyzing Student Academic Performance, Gender-wise Trends, Subject Scores, and Parental Education Impact.
**Short Description**
Developed an interactive Power BI Student Performance Analytics Dashboard to analyze student scores in Math, Reading, and Writing. The dashboard includes KPI cards, gender-wise analysis, parental education insights, and interactive filters to support data-driven decision-making and performance evaluation.
**Tech Stack**
Power BI Desktop
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions) (Measures & Calculations)
Microsoft Excel / CSV (Dataset)
Data Modeling
**Data Source:** 
Kaggle – Students Performance in Exams Dataset (CSV), containing student demographics, parental education, ethnicity, and Math, Reading, and Writing scores
**Features**
Interactive KPI Cards (Total Students, Average, Maximum & Minimum Scores)
 Subject-wise Performance Analysis (Math, Reading, Writing)
Gender-wise Score Comparison
 Parental Education Level Analysis
Ethnicity-based Data Filtering
